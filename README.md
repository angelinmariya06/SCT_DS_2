# SkillCraft Technology - Task 2

## 📌 Objective

Perform data cleaning and exploratory data analysis (EDA) on the Titanic dataset to explore relationships between variables and identify patterns and trends.

## 📂 Dataset

- Titanic Dataset (Kaggle)

## 🛠️ Tools & Libraries Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📋 Tasks Performed

- Loaded the Titanic dataset.
- Explored the dataset using `head()`, `shape()`, `info()`, and `describe()`.
- Checked for missing values.
- Filled missing values in the **Age** column using the median.
- Filled missing values in the **Embarked** column using the mode.
- Removed the **Cabin** column due to a large number of missing values.
- Performed exploratory data analysis (EDA).
- Visualized the data using different charts.
- Explored relationships between variables.
- Identified patterns and trends from the dataset.

## 📊 Visualizations

- Survival Distribution
- Gender Distribution
- Age Distribution
- Passenger Class Distribution
- Survival by Gender
- Survival by Passenger Class
- Age Distribution by Survival Status
- Fare Distribution by Passenger Class (Box Plot)
- Correlation Heatmap

## 🔍 Key Findings

- Female passengers had a higher survival rate than male passengers.
- First Class passengers had better survival rates than Second and Third Class passengers.
- Most passengers were between 20 and 40 years of age.
- Third Class had the highest number of passengers.
- First Class passengers generally paid higher fares.
- Passenger Class and Fare showed a strong negative correlation.
- Most numerical variables had weak correlations, while **SibSp** and **Parch** showed a moderate positive correlation.
